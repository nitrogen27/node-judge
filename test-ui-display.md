# Тесты для проверки UI

## Проблема
В UI отображается 0.00Mb для памяти, хотя сервер измеряет память корректно.

## Что было исправлено:

1. **Сервер (server.js)**:
   - Исправлена проверка `if (res.memMb && res.memMb > maxMemoryMb)` на `if (res.memMb !== undefined && res.memMb !== null && res.memMb > maxMemoryMb)`
   - Добавлено логирование отправляемых данных через WebSocket
   - Улучшена логика сбора максимальной памяти из результатов тестов

2. **Клиент (App.js)**:
   - Добавлено логирование в консоль для отладки
   - Добавлены проверки на наличие данных summary
   - Добавлены предупреждения, если память равна 0

## Как проверить:

1. Откройте браузер и консоль разработчика (F12)
2. Отправьте решение через UI
3. Проверьте консоль на наличие сообщений:
   - `[Client] Получено сообщение: results`
   - `[Client] Результаты получены: { maxMemoryMb: ... }`
   - `[Client] ✓ Память корректна: X.XX MB`

4. Если в консоли видно корректное значение, но UI показывает 0, проблема в отображении
5. Если в консоли видно 0 или undefined, проблема в передаче данных

## Команды для тестирования:

```bash
# Тест измерения памяти
node test-server-direct.js

# Тест передачи через WebSocket API
node test-websocket.js

# Интеграционный тест WebSocket
node test-ui-integration.js
```
